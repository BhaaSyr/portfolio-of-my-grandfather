# Literary Archive - Dr. Bahauddin Abdullah Al-Zohouri

هذا المشروع هو موقع ويب مبني باستخدام **WordPress**، يهدف إلى إنشاء أرشيف رقمي شامل لجميع الأعمال الأدبية للدكتور **بهاء الدين عبد الله الزهوري**، وعرضها بشكل منظّم ومتاح للقرّاء والباحثين والمهتمّين بالأدب العربي.

## فكرة المشروع
انطلق هذا المشروع من الحاجة إلى:
- حفظ الأعمال الأدبية من الضياع
- جمعها في مكان واحد موثوق
- تقديمها للقرّاء بصيغة رقمية سهلة الوصول

## المحتوى
- عرض الأعمال الأدبية بشكل منظّم
- صفحات تعريفية بالسيرة الذاتية
- إمكانية تحميل النسخ الأصلية (PDF) لبعض الأعمال
- تصميم بسيط يركّز على المحتوى
- دعم كامل للغة العربية (RTL)


## ملاحظة
هذا المستودع مخصّص لعرض فكرة المشروع ونتيجته النهائية، ولا يحتوي على نواة ووردبريس أو ملفات الحسّاسة.
